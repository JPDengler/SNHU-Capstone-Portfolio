Overview

The Nuoro Weight-Tracking App (on main Branch or Enhancement 2 Branch) was improved to enhance data visualization, user interactivity, and validation. This enhancement focused on improving the user experience, ensuring data reliability, and introducing dynamic visual feedback.

Key Improvements

  • Sorting Options: Added functionality to sort weight entries by Date (Ascending/Descending) and Weight (Ascending/Descending).
  • Search Feature: Allowed filtering of entries by date range and weight range.
  • Optimized Database Queries: Improved SQLite queries for faster and more efficient retrieval.
  • Bug Fixes: Resolved issues with disappearing charts and RecyclerView scrolling errors.

Technical Implementation

The implementation involved changes to the following areas:

  • Modified the DatabaseHelper class to support sorting and searching capabilities using SQL queries.
  • Enhanced the DataDisplayActivity to integrate user input for sorting and filtering functionality.
  • Refactored the graphing logic to dynamically update based on sorting and filtering operations.
  • Introduced input validation to ensure that users enter valid date ranges and weight thresholds.
  • Fixed RecyclerView scrolling bugs to ensure all entries are viewable regardless of dataset size.
