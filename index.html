Enhancement One: Nuoro Weight Management App

The Nuoro Weight Management App was designed as a capstone project to address real-world health management needs. This application allows users to:

  • Log daily weight entries with timestamps.
  • Set and track weight goals.
  • Edit and delete previous entries.
  • Receive SMS notifications upon reaching their goals (with user permission).

The app employs a structured database with three primary tables: users, daily weight entries, and goal weight. It features a clean user interface with interactive elements, ensuring accessibility and ease of use.
