Overview

View Repository

The Nuoro Weight-Tracking App is a mobile application designed to help users track their weight, set personal goals, and visualize progress over time. Developed originally in CS-360: Mobile Architect & Programming, the app underwent significant upgrades in Enhancement One as part of the CS-499: Computer Science Capstone.

These enhancements focused on implementing industry-standard solutions, improving user interactivity, and ensuring secure, reliable data handling. Major improvements included integrating RecyclerView for dynamic data management, MPAndroidChart for graphical data visualization, and addressing SMS permission handling for secure notifications.

Full Documentation

Key Improvements

  • Input Validation: Ensured valid date format (MM/DD/YY) and weight range (1-500 lbs) to maintain data integrity.
  • Interactive Data Management: Replaced static TextView with RecyclerView to allow users to edit and delete individual weight entries.
  • Data Visualization: Integrated MPAndroidChart to dynamically display weight trends with a goal weight marker.
  • Improved SMS Permissions: Fixed broken request flows, ensuring proper handling of Android's permission model.
  • User Feedback Enhancements: Added detailed feedback for actions, including success messages, error dialogs, and confirmation prompts.
  • Database Enhancements: Refactored DatabaseHelper for real-time updates and robust error handling.
  • Goal Weight Management: Validated goal weight inputs and displayed the goal weight on the trend chart.

Technical Implementation

The following changes were implemented to enhance the app:

  • Refactored the DatabaseHelper class to validate inputs and handle real-time updates for a smoother user experience.
  • Integrated MPAndroidChart to provide a dynamic and visually appealing way to track weight progress over time.
  • Enhanced the RecyclerView to support intuitive data management with edit and delete actions, making the app more interactive.
  • Implemented detailed user feedback mechanisms, including error messages and success notifications, to enhance usability.
