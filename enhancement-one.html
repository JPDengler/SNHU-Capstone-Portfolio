Overview

This enhancement improved the Nuoro Weight-Tracking App by introducing data visualization, user interactivity, and validation to enhance the experience of weight tracking.

Key Improvements

  • RecyclerView: Replaced static weight logs with a dynamic, scrollable list.
  • MPAndroidChart: Integrated real-time graphing of weight progress.
  • Input Validation: Ensured users enter realistic weight values (1-500 lbs).
  • SMS Notifications: Implemented permission handling for weight goal alerts.
